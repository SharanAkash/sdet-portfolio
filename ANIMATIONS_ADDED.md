## 🎯 Animation Types:

### **Float Animation:**
- Icons gently move up and down
- Rotation for dynamic feel
- Different speeds (5-9 seconds)
- Staggered delays for natural look

### **Slide Animation:**
- Code lines move left/right
- 15-18 second duration
- Fade in/out at edges
- Creates motion effect

### **Pulse Animation:**
- Subtle scale changes
- Used for checkmarks
- Represents test execution

### **Rotate Animation:**
- Slow 360° rotation (25-30 seconds)
- Used for gears and code symbols
- Represents continuous integration

## 🔧 Technical Details:

### **CSS Animations Used:**
1. `floatUpDown` - Vertical floating motion
2. `slideRight` - Horizontal movement (left to right)
3. `slideLeft` - Horizontal movement (right to left)
4. `checkmark` - Scale and position change
5. `bugFloat` - Complex 3D-like movement
6. `pulse` - Subtle scaling
7. `rotate` - 360° rotation
8. `gradient` - Background color shift

### **Colors:**
- Primary: `#6366f1` (Indigo)
- Secondary: `#ec4899` (Pink)
- Accent: `#14b8a6` (Teal)
- Warning: `#f59e0b` (Amber)

### **Performance:**
- GPU acceleration enabled
- Will-change properties for smooth animation
- Requestanimationframe compatible
- No layout reflows
- Optimized for 60fps
